# 03 Dpo Preference Alignment Project

This project is self-contained for Colab demo.

## Contents
- `stage3_dpo_preference_alignment.ipynb` - stage notebook with training, evaluation, and ipywidgets UI.
- `stage3_data.zip` - bundled dataset ZIP. The notebook automatically extracts this file into `data/`.

## How to run in Colab
1. Upload both `stage3_dpo_preference_alignment.ipynb` and `stage3_data.zip` to the same Colab working directory, or upload/extract this project ZIP.
2. Open the notebook.
3. Run cells top to bottom.
4. The extraction cell will create the `data/` folder automatically.

No individual dataset file upload is required.
