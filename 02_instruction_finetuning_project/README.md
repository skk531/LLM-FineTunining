# 02 Instruction Finetuning Project

This project is self-contained for Colab demo.

## Contents
- `stage2_instruction_finetuning.ipynb` - stage notebook with training, evaluation, and ipywidgets UI.
- `stage2_data.zip` - bundled dataset ZIP. The notebook automatically extracts this file into `data/`.

## How to run in Colab
1. Upload both `stage2_instruction_finetuning.ipynb` and `stage2_data.zip` to the same Colab working directory, or upload/extract this project ZIP.
2. Open the notebook.
3. Run cells top to bottom.
4. The extraction cell will create the `data/` folder automatically.

No individual dataset file upload is required.
